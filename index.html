    • Musicplayer

    jquery musicPlayer plugin

    Project maintained by ernesto27

    Install

    put styles in the header

    load javascript files above the closing body tag

    Examples

      $(document).musicPlayer({
        sourceAudio: [
            { src : "audio/song1.mp3"},
            { src : "audio/song2.mp3"},
        ],
        autoplay:true,
        loop: true,
        position: "#music-player" // default bottom of the page

    });

    Options

    autoplay -> Play after load page

    loop -> loop songs
